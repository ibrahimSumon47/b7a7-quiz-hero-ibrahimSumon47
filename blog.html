Q1: Difference between map, filter, forEach and find

map: When I want to transform elements in an array.

filter: When I want to select a subset of multiple elements from an array.

forEach: It is a generic tool only use it when we can't use a more specialized tool.

find: When I want to select a single element from an array.

Q2: Difference between Local storage, Session storage and Cookies

Local storage: It is useful for storing data that the user will need to access later, such as offline data. It support HTML5.

Session storage: It is a great way to improve the performance of your web application but session storage can't be use in offline. Session storage also support HTML5.

Cookies: It is a good choice for storing data that should not be persisted for a long time such as session IDs. But cookies support oldest HTML4.

Q3: What is eventLoop?

eventLoop: It is like the simplest little piece in the equation and it has one very simple job. The event loops job is to look at the stack and look at the task queue. If the stack is empty it takes the first thing on the queue and pushes it on to the stack which effectively run it.
